# 🪖 坦克大战网页版

[在线体验](https://tank.qizhen.xyz/)

一个基于 **HTML5 Canvas** 的硬核复刻项目。本项目不仅还原了经典《坦克大战》的逻辑，更深度集成了变态 AI 与道具反制机制，并采用现代前端工程化标准构建。

## 🌟 核心工程亮点

* **🚫 零资源依赖 (Zero-Asset Architecture)**: 所有视觉元素（坦克、地形、特效）均采用程序化绘图（Canvas API）渲染，甚至音效也由 **Web Audio API** 实时合成。
* **🛠 确定性碰撞系统**: 针对 $20 \times 20$ 像素网格实现的 AABB 矩形碰撞检测，并通过半网格（10px）对齐逻辑优化转向体验。
* **🧠 智能 AI 引擎**:
    *   **A* 寻路**: 动态计算最优路径，支持破墙代价评估。
    *   **策略锁定 (Strategy Commitment)**: 锁定目标（基地、玩家或道具）持续 3-5 秒，避免目标反复横跳。
    *   **兜底逻辑**: 若基地无法到达（如被钢墙包围），AI 会自动切换为“猎杀玩家”模式。
* **🔊 音效合成引擎**: 摒弃传统的 MP3/WAV，使用振荡器（Oscillator）模拟 8-bit 处理器的方波 and 噪声效果。

## 🎮 交互指南

### 操作方案

| 维度 | 键盘控制 | 触控 / 鼠标 / 触摸控制 |
| --- | --- | --- |
| **移动** | `W/A/S/D` 或 `↑↓←→` | 坦克区域内**拖拽** |
| **射击** | `Space` 或 `Z` | 坦克区域外**点击** |
| **策略** | `P` (暂停) | **点击目标方向** 自动转向并连续射击 |
| **调试** | `Ctrl + D` | **切换路径显示** (显示坦克预设轨迹) |

### 🛠️ 调试模式 (Debug Mode)
按下 `Ctrl + D` 可实时查看敌方坦克的逻辑：
- **彩色虚线**: 每辆坦克拥有独立颜色的路径线，展示其 A* 算法生成的预设轨迹。
- **目标标记**: 路径终点带有圆形标记，代表该坦克当前锁定的终极目标。

## 🎨 地图编辑器 (Map Editor)

项目集成了一个功能强大的可视化地图编辑器，方便开发者快速创作关卡。

*   **访问地址**: `/editor.html`
*   **核心功能**:
    *   **可视化绘图**: 支持单点切换地形或按住鼠标左键**批量拖拽绘画**。
    *   **全地形支持**: 预设砖墙、钢墙、森林、水面、冰面及基地，支持一键**橡皮擦**。
    *   **剪贴板数据流**:
        *   **复制数据**: 一键导出 $30 \times 38$ 矩阵字符串，直接粘贴至 `src/world/levels/levelXX.txt`。
        *   **加载数据**: 直接通过剪贴板读取数据，实现无缝二次编辑。
    *   **现代 UI**: 采用 **Glassmorphism (玻璃拟态)** 设计风格。

## 🏗 技术架构 (System Architecture)

项目采用解耦的**系统驱动设计**，通过各子系统协同工作：

* **GameManager**: 核心控制器，管理 `BOOT` -> `MAIN_MENU` -> `STAGE_INTRO` -> `PLAYING` -> `STAGE_CLEAR` -> `SCORE_TALLY` -> `GAME_OVER` 的状态转移。
* **CollisionSystem**: 处理坦克、子弹、地形间的精细碰撞逻辑及实体间的排斥分离。
* **PowerUpSystem**: 管理道具生成、生存周期以及玩家/敌方拾取后的正/负反馈效应。
* **SoundManager**: 基于 Web Audio API 的实时音效合成引擎。

## ⚙️ 核心逻辑规格

### 1. 地形系统 (Terrain Types)

| ID | 名称 | 特性描述 |
|---|---|---|
| **1** | 砖墙 (Brick) | 可被子弹摧毁，阻挡移动。 |
| **2** | 钢墙 (Steel) | 无法摧毁（除非满级火力），阻挡移动。 |
| **3** | 森林 (Forest) | 遮挡坦克与子弹视线，不阻挡移动。 |
| **4** | 水面 (Water) | 阻挡坦克移动（需特殊能力），不阻挡子弹。 |
| **5** | 冰面 (Ice) | 坦克在其上行走会产生惯性滑动（约 20 帧）。 |
| **6** | 基地 (Base) | 2x2 核心，被击中即告游戏失败。 |

### 2. 道具表 (Power-Ups)

| 图标 | 名称 | 玩家拾取效果 | 敌方拾取效果 |
|------|------|-------------|-------------|
| ⭐ | 星星 (Star) | 坦克等级 +1 (上限 4 级) | 玩家等级 -1 |
| 🔫 | 手枪 (Gun) | 升至最高等级 (ARMOR) | 全场敌军等级 +1 |
| ❤️ | 1UP (Tank) | 生命值 +1 | 玩家生命值 -1 |
| ⛏️ | 铲子 (Shovel) | 基地围墙变钢 20 秒 | 销毁基地周围围墙 |
| 🛡️ | 头盔 (Helmet) | 无敌护盾 (10 秒) | 移除玩家当前护盾 |
| ⏰ | 时钟 (Clock) | 敌方冻结 (10 秒) | 全场敌军永久加速 |
| 💣 | 炸弹 (Bomb) | 消灭当前全场敌军 | 玩家受到 1 次伤害 |

### 3. 物理规格

* **画布总尺寸**: $600 \times 800$ 像素（含 HUD）。
* **战场中心**: $30 \times 38$ 网格。
* **基础单元格 (Tile)**: $20 \times 20$ 像素。
* **坦克尺寸**: $40 \times 40$ 像素（占 2x2 网格）。
* **碰撞容错**: $\pm 10$ 像素半网格自动对齐。
